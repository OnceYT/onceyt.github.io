01

Overview

One is a general-purpose Discord bot offering server information commands, moderation tools, light utility features, and a voice-channel music player. It operates entirely through Discord's slash commands and only acts within servers it has been added to.

02

Information we collect

The bot receives the following from Discord in the normal course of operating:

  • Identifiers — your Discord user ID, username, and server (guild) ID, needed to run commands and address responses correctly.
  • Server data — role, channel, and membership information made available by Discord, used to display info commands and enforce permission checks.
  • Command input — text you voluntarily provide to a command, such as an AFK reason, or a reason a moderator enters when issuing a warning or ban.
  • Status & activity — your current Discord status and activity, only when another member looks it up (see §04).
    • 03

    What gets stored, and where

    Most information the bot touches is used live and immediately discarded — for example, the data shown by /serverinfo or /userinfo is never saved anywhere. The bot does persist a small set of moderation records to a local data file on the server that runs the bot (not on Discord's own infrastructure):

  • Warning records: user ID, reason, issuing moderator, and timestamp.
  • Temporary ban records: user ID, reason, and scheduled unban time.
  • A server's configured moderation log channel ID, if one has been set.
    • This data is not shared, sold, or transmitted to any third party. It exists solely so moderators in your server can track warning and ban history, and is only ever accessible to that server's moderators through the bot's own commands.
    04

    Status & activity data

    Commands like /activity and /userinfo display a member's current Discord status and what they're doing (for example, a game or Spotify track) at the moment the command is run. This information is rendered directly into the response and is never written to a file or database.

    Any member able to use these commands can currently look up another member's live status. There is no per-user opt-out at this time; if you'd prefer your activity not be visible this way, you can adjust your own Discord status visibility settings, or ask a server's administrators to restrict access to these commands.

    05

    Message content

    The bot does not read, log, filter, or act on the content of ordinary messages sent by server members. No feature in the bot processes message text for regular users.

    A separate, owner-only diagnostic tool is used for maintenance — it lets the bot's developer run debugging commands while the bot is live. This is restricted to the registered bot owner account and is not exposed to, or triggered by, other members' messages.

    06

    How information is used

    Information is used only to operate the bot's commands and moderation features as described above — there is no advertising, profiling, or analytics use of any data the bot handles.

    07

    Retention & deletion

    Warning and temporary-ban records are kept until a server's moderators remove them using the bot's own commands, or until the bot is removed from that server. To request removal of records concerning you, contact a moderator in the relevant server, or reach out using the details below.

    08

    Third parties & AI training

    No data the bot collects is sold, shared with advertisers, or transferred to third parties. None of it is used to train machine learning or AI models.

    09

    Children's privacy

    Discord requires users to be at least 13 years old (or the minimum age in their region). The bot does not knowingly collect information beyond what Discord itself provides for any user, and does not target or market to children.

    10

    Changes to this policy

    This policy may be updated as the bot's features change. The effective date at the top of this page will be revised whenever that happens. Continued use of the bot after a change constitutes acceptance of the updated policy.
